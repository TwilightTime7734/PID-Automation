# Visual Studio Community Requirements

This file lists the required setup to open, build, and run `DronePidTuningAssistant.WinForms` in Visual Studio Community.

## Required Software

1. Visual Studio Community 2022 (17.8 or newer recommended)
2. .NET 8 SDK (or newer SDK that supports `net8.0-windows`)

## Required Visual Studio Workload

1. `Desktop development with .NET`

## Required Components (Individual Components tab)

1. `.NET SDK 8.0` (or newer)
2. `.NET desktop development tools`
3. `Windows Forms .NET` tooling

## Optional but Recommended

1. Git for Windows
2. Latest Visual Studio updates

## Open and Run

1. Open `DronePidTuningAssistant.WinForms.csproj` in Visual Studio Community.
2. Set configuration to `Debug` and target `Any CPU` (default is fine).
3. Press `F5` to run.
